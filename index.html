About Me

連結技術與管理的橋樑。

我的學習旅程從工業管理出發，逐漸深入到資訊科學的領域。透過學習等候理論與統計學，我具備了發現系統瓶頸、分析數據的能力；而透過 C++ 與 Java 的扎實訓練，我擁有了將解決方案實作出來的技術力。

在過去的專案中，我曾運用工業工程的知識分析校園周邊早餐店的排隊流程，也曾從零開始實作經典的 C++ 貪食蛇遊戲。我認為最好的工程師不僅懂得寫出乾淨的程式碼，更懂得這段程式碼能為真實世界帶來什麼價值。
